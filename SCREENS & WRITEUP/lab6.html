CMPSC 457 Lab #6

By: Matthew Panetta

Instructions

In this lab I needed to:
  • Draw an environment consisting of two tables
  • Move the camera and walk around in the environment
  • Add an overhead light and three lamps (red, green, blue) to the scene.

My Implementation

  • I made the two tables using scaled cubes. The lamps are made of a cube base, an elongated cube support, and the lamp is a wire sphere.
  • The overhead light emits diffuse, ambient, and specular light. The user can control the light intensity using the 0-9 keys.
  • The user can move around the scene using the arrow keys. UP = move forward, DOWN = move backward, LEFT = change heading left, RIGHT = change heading right. PAGE UP = ascend vertically, PAGE DOWN = descend vertically. HOME = look up, END = look down.
  • The three lamp lights emit red, green, and blue diffuse light respectively.
  • The lamp heads have colored material, everything else has gray material.
